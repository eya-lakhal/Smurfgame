# Smurfgame
 SmurfGame - La Forêt Magique Jeu Windows Forms développé en C# avec une architecture multicouche. Un Schtroumpf se déplace dans une forêt magique et doit éviter des ennemis (araignées, mouches, insectes). 
## Dépendances

Ce projet utilise :

* DataLayer
* UserControlLayer

### Installation :

1. Cloner les projets :

   * SmurfGame
   * DataLayer
   * UserControlLayer

2. Ouvrir SmurfGame dans Visual Studio

3. Ajouter les projets :

   * Clic droit sur Solution → Add → Existing Project
   * Ajouter DataLayer et UserControlLayer

4. Refaire les références si nécessaire

5. Lancer le projet
